Main
====
Ctrl+O          Browse for folder
Ctrl+S          Save all
F5              Refresh
BwRfsh          Refresh
Alt+Right       Select next
Alt+Shift+Right Select next (do not focus Object Name)
Alt+Left        Select previous
Alt+Shift+Left Select previous (do not focus Object Name)
BwBack          Navigate backward
BwFwd           Navigate forward
BwHome          Go to My Pictures
BwStop          Stop loading miniatures

Main - in folder list only
--------------------------
Backspace       Go up

Main - in image list when in TC mode only
-----------------------------------------
Ctrl+A          Select all
Space           Toggle selection of focused item
Insert          Toggle selection of focused item and move to next
Shift+Insert    Toggle selection of focused item and move to previous
Left            Focus previous item
Right           Focus next item
Up              Focus item up
Down            Focus item down
PageUp          Focus item page up
PageDown        Focus item page down
Home            Focus first item
End             Focus last item
*               Toggle selection of all items

Preview
=======
Left            Select previous
Right           Select next
Enter           Toggle full screen
Esc             Close
L               Rotate left
R               Rotate right

KeyWordsEditor (keyword list)
============== --------------
Ctrl+C          Copy selected keywords
Ctrl+X          Cut selected keywords
Ctrl+V          Paste
Del             Delete selected keywords
