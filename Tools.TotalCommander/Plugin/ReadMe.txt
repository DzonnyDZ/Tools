The Tools.TotalCommander.Plugin project is template project for building Total Commander plugin wrappers.
This project is not intended to be build separatelly (or as part of solution build).
Files of this project are used as template files by TotalCommander Plugin Builder. Either from file system or from copy of this project zipped and embededed into TC Plugin Builder assembly.
Selected files are used by wrapper as-is while other files are overwritten by building process.
